Just run the code and it will bring you to the Triva game!

The program will welcome you to the game, ask the users name and store the users name for later use.
The first question will be asked and the user can pick from a few different answers.
When the user picks the right answer they get one point and the program gives them a positive reply.
If you get it wrong, the program will minus a point and let you keep trying to answer and minusing points until
the user gets the answer right.
At the end of the questions it tells you the total amount of points.
